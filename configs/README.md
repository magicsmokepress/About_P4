# Board Configuration Files

Drop the appropriate `board_config.h` into your sketch folder (alongside your `.ino` file).

| File | Board |
|------|-------|
| `board_config_crowpanel_7inch.h` | Elecrow CrowPanel Advanced 7" (primary — Chapters 4–25) |
| `board_config_waveshare_p4eth.h` | Waveshare ESP32-P4-ETH (Chapters 32, 33, 35) |
| `board_config_m5stack_tab5.h` | M5Stack Tab5 (Chapters 26, 29, 30, 31) |

Rename the file you need to `board_config.h` in your sketch folder.
See Appendix A in the book for full hardware notes.
